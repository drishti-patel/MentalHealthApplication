# 🧠 Mental Health Prediction App

An AI-powered web application that analyzes user-entered text and predicts potential mental health conditions using Natural Language Processing (NLP) and Machine Learning models, including RoBERTa and Logistic Regression.

## Features

* Real-time text analysis
* Mental health prediction from user input
* Confidence score display
* Responsive React-based interface
* Flask backend with NLP models

## Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Flask, Python
* **ML/NLP:** RoBERTa, Logistic Regression, Scikit-learn, Transformers

## Getting Started

```bash
# Backend
pip install -r requirements.txt
python app.py

# Frontend
npm install
npm start
```
